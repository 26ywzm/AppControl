应用控制器组件示例

这个示例展示了如何在任何网页中嵌入应用控制器组件。

应用控制器

如何在其他页面中使用

1. 引入控制器脚本：

<script src="appControlWidget.js"></script>

2. 创建容器元素：

<div id="yourContainerId"></div>

3. 初始化控制器：


const appWidget = new AppControlWidget({
    serverUrl: 'http://your-server-ip:3001'  // 替换为你的服务器地址
});
appWidget.init('yourContainerId');
